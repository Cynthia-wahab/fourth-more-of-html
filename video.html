#To add a audio we need to write audio tag. It's almost same as img tag. only need to add "controls" in it. Example:

<audio controls src="media file"></audio>

#To add a video we need to write video tag. It's almost same as img tag. only need to add "controls" in it. Example:

<video controls src="media file"></video>

We can also use video from other website. To do that, go the the website copy the embeded link and put it in p tag.
